
1.  В JS типы классифицируются на преметивные типы данных, на спецальное значение 'undefined' и 'null'
    a также "Object"

2. К преметивным типа данных относится типы:
   "Number"
   "BigInt"
   "String"
   "Boolean"
   "Symbol"

3. "NaN" относится к типу 'Number'
   "Infinity" относится к типу 'Number'

4. В 'BigInt' можно присваивать только челые числа. Числа могут быть как положительными так и отрицательными.
     'BigInt' совместим только с 'BigInt'
    значения относяшиеся к типы  'BigInt' может в себе содержать огромные числа.

5. Тип данных 'Number' позволяет в себе хранить числа с плавующей точкой.

6. Строки в JS должны быть заключены в '', "" или `` кавычки
   ` этот символ называется тильдой
   эту нужно для того что-бы интерпретатор распознал тип данных 'String'  в коде программы.

7. 'Boolean' тип данных имеит всего два значения 'false' и 'true'
   'false' и 'true' это значения логического типа
    присравнинии мы получаем логический тип данных

8. let amount;
   amount      // undefined
   Если объявить переменную без присвоение ей какогото значение то её значение будет равно объекту 'undefined'

9. В JS  функция по умолчанию возвращает значение 'udefined'.

10. 'undefined' используем для неизвестных значение которые относятся к преметивным типа данных
    'null'      используем для неизвестных значений которые относятся к объектному типу
