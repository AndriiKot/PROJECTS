
1. Дан объект 'values' как можно пробежаться по полям этого объекта?
Примените к каждому значению поля операцию 'typeof' и выедете результат с
новой строки для каждого значения поля.

const values = {
	name: 'Mouse',
	price: 25,
	stock: false,
	avg: (a, b) => (a + b) / 2n,
	reference: null,
	primitive: undefined,
	numbers: [NaN, Infinity],
}
